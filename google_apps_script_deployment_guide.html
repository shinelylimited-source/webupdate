Google Apps Script Deployment and Integration Guide

This guide provides step-by-step instructions on how to deploy your Google Apps Script for handling form submissions and integrate it with your index.html file. This setup will allow your website's contact form to send emails directly to your specified address.

Step 1: Prepare Your Google Apps Script Code

You should have a Google Apps Script file (e.g., google_apps_script.js) similar to the one provided previously. Ensure you have replaced the placeholder email address with your actual recipient email.

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    
    var name = data.name;
    var email = data.email;
    var phone = data.phone || 'Not provided';
    var message = data.message;
    
    var recipient = 'your-email@example.com'; // <strong>CHANGE THIS TO YOUR EMAIL</strong>
    var subject = 'New Contact Form Submission: ' + name;
    
    var body = 'You have a new message from your website contact form:\n\n' +
               'Name: ' + name + '\n' +
               'Email: ' + email + '\n' +
               'Phone: ' + phone + '\n\n' +
               'Message:\n' + message;
    
    MailApp.sendEmail({
      to: recipient,
      subject: subject,
      replyTo: email,
      body: body
    });
    
    return ContentService.createTextOutput(JSON.stringify({ 'result': 'success' }))
      .setMimeType(ContentService.MimeType.JSON);
      
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ 'result': 'error', 'error': error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

Important: Make sure to replace 'your-email@example.com' with the actual email address where you want to receive form submissions.

Step 2: Deploy the Google Apps Script as a Web App

  1. Go to script.google.com.
  2. If you don't have an existing project, click New project.
  3. Copy the entire content of your Google Apps Script file (from Step 1) and paste it into the code editor, replacing any default content (like function myFunction() {}).
  4. Save the project (File > Save, or Ctrl+S/Cmd+S). You might be prompted to give your project a name.
  5. Click the Deploy button (usually a blue button on the top right) and select New deployment.
  6. In the "Select type" dropdown, choose Web app.
  7. Configure the deployment settings:
    • Description: (Optional) Add a brief description, e.g., "Contact Form Email Sender".
    • Execute as: Select Me (your Google account email). This ensures the script runs with your permissions to send emails.
    • Who has access: Select Anyone. This allows your website to send requests to the script without requiring users to log in to a Google account.
  8. Click Deploy.
  9. The first time you deploy, Google will ask you to authorize the script. Click Authorize access, select your Google account, and grant the necessary permissions (e.g., to send emails as you).
  10. After successful deployment, you will see a dialog with a Web app URL. Copy this URL. This is your unique endpoint for the form submission.

Step 3: Integrate the Web App URL into Your index.html

Now, you need to update your index.html file to point its form submission logic to the deployed Google Apps Script Web App.

  1. Open your index.html file in a text editor.
  2. Locate the JavaScript section that handles the contact form submission. You'll be looking for a line similar to this:
    3. const scriptURL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';
  3. Replace 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE' with the Web app URL you copied in Step 2.
    4. // Example of updated JavaScript in your index.html
if (contactForm) {
  contactForm.addEventListener('submit', (e) => {
    e.preventDefault();

    // ... (form validation logic)

    if (isValid) {
      const originalBtnText = submitBtn.textContent;
      submitBtn.textContent = 'Sending...';
      submitBtn.disabled = true;

      // <strong>REPLACE THIS URL WITH YOUR DEPLOYED WEB APP URL</strong>
      const scriptURL = 'https://script.google.com/macros/s/AKfycbz_YOUR_UNIQUE_ID_HERE/exec'; 

      fetch(scriptURL, {
        method: 'POST',
        mode: 'no-cors', 
        cache: 'no-cache',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({
          name: name,
          email: email,
          phone: phone,
          message: message
        })
      })
      .then(() => {
        formSuccess.classList.add('show');
        contactForm.reset();
        formSuccess.scrollIntoView({ behavior: 'smooth', block: 'nearest' });
      })
      .catch(error => {
        console.error('Error!', error.message);
        alert('There was an error sending your message. Please try again later.');
      })
      .finally(() => {
        submitBtn.textContent = originalBtnText;
        submitBtn.disabled = false;
      });
    }
  });
}

Step 4: Test Your Contact Form

Open your index.html file in a web browser and fill out the contact form. Submit the form and check your designated email inbox to ensure you receive the message. If you encounter any issues, double-check the following:

  • The Web App URL in your index.html is correct and matches the one from your Google Apps Script deployment.
  • The recipient email in your Google Apps Script is correct.
  • The Google Apps Script is deployed with "Execute as: Me" and "Who has access: Anyone".
  • You have authorized the script to send emails.

By following these steps, your website's contact form will now be fully functional, sending messages directly to your email via Google Apps Script.
